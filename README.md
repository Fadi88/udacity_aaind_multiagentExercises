# Udacity AAIND Multi-Agent Exercises

This repository contains exercises for the Udacity AI Agentic Interface Nanodegree (AAIND) Multi-Agent Systems course.

## Setup

1.  Install `uv`:
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```

2.  Install dependencies:
    ```bash
    uv sync
    ```

3.  Set up environment variables:
    ```bash
    cp .env.example .env
    # Edit .env with your API keys
    ```
